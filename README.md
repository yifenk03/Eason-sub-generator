# 字幕生成工具 (VAD分段+SenseVoice GPU版)

基于 FunASR 的智能字幕生成工具，采用 **VAD分段 + SenseVoice识别** 架构，支持 GPU 加速，可自动将视频转录为带精确时间戳的 SRT/ASS 字幕文件。

## 核心特性

- **VAD 智能分段**：使用 fsmn-vad 模型检测语音段落，确保每个字幕条目对应一个完整的语音片段
- **保留 SenseVoice**：继续使用 SenseVoiceSmall 模型进行语音识别
- **GPU 加速**：支持 NVIDIA CUDA 加速，大幅提升处理速度
- **多格式输出**：支持 SRT 和 ASS 两种字幕格式
- **智能校对**：集成 LM Studio LLM 模型，自动优化字幕文本
- **可调参数**：支持自定义静音间隔阈值和最小语音时长

## 工作流程

```
视频文件 → VAD模型分段 → 逐段SenseVoice识别 → LLM校对 → 字幕输出
              ↓
         精确时间戳
```

## 系统要求

### 硬件要求
- NVIDIA 显卡（推荐 4GB 以上显存，支持 CUDA）
- 如不使用 GPU，至少需要 8GB 以上内存

### 软件要求
- Python 3.8+
- FFmpeg（用于音频提取）

## 安装步骤

### 1. 安装 Python 依赖

```bash
pip install -r requirements.txt
```

### 2. 下载 FunASR 模型

工具使用 SenseVoiceSmall 模型，首次运行会自动下载。如需手动配置：

```python
# 在 short-mt-gpu.py 中修改模型路径
MODEL_DIR = r"你的模型路径\SenseVoiceSmall"
```

### 3. 安装并启动 LM Studio

1. 下载并安装 [LM Studio](https://lmstudio.ai/)
2. 下载适合的校对模型（如 Qwen、Llama 等中文模型）
3. 启动 LM Studio 并加载模型，确保 API 服务运行在 `http://127.0.0.1:1234`

## 使用方法

### 启动工具

```bash
python short-mt-gpu.py
```

### 操作步骤

1. **GPU 设置**：勾选"启用GPU加速"，选择 GPU 设备号
2. **输入设置**：点击"选择文件"或"选择文件夹"添加视频
3. **输出设置**：
   - 设置保存位置
   - 选择字幕格式（SRT 或 ASS）
4. **VAD 分段设置**（关键）：
   - **静音间隔阈值**：句子之间的静音时长，默认 0.3 秒
   - **最小语音时长**：过滤短噪音片段，默认 0.1 秒
5. **处理设置**：根据硬件配置调整并行线程数
6. **LM Studio 设置**：确保已连接并选择校对模型
7. **校对提示词**：可使用默认提示词或根据需求调整
8. 点击"开始生成字幕"

### VAD 参数调节建议

| 场景 | 静音间隔阈值 | 最小语音时长 |
|------|-------------|-------------|
| 正常语速演讲 | 0.3s | 0.1s |
| 快节奏对话 | 0.2s | 0.1s |
| 慢速朗读 | 0.5s | 0.2s |
| 有背景音乐 | 0.4s | 0.15s |

## 字幕格式说明

### SRT 格式
标准字幕格式，兼容所有播放器：
```
1
00:00:01,000 --> 00:00:03,500
这是第一条字幕

2
00:00:04,200 --> 00:00:06,800
这是第二条字幕
```

### ASS 格式
高级字幕格式，支持样式和特效：
- 可自定义字体、大小、颜色
- 支持文字特效和定位
- 推荐用于有定制需求的用户

## 处理流程详解

1. **VAD 语音检测**
   - 使用 fsmn-vad 模型分析音频
   - 检测每个语音片段的开始和结束时间
   - 过滤静音和短噪音片段

2. **逐段语音识别**
   - 对每个 VAD 检测到的语音片段
   - 提取对应的音频数据
   - 使用 SenseVoice 进行识别
   - 保留原始时间戳信息

3. **LLM 校对**
   - 合并所有识别文本
   - 发送到 LM Studio 进行校对
   - 使用【SPLIT】分隔符切分结果

4. **字幕生成**
   - 将校对后的文本分配回对应时间戳
   - 生成 SRT 或 ASS 格式文件

## 常见问题

### Q: VAD 检测不到语音？
- 检查音频是否清晰，有无背景噪音
- 尝试调低"最小语音时长"参数
- 确认视频文件编码正确

### Q: 生成的字幕时间轴不准？
- VAD 分段本身是准确的
- 如果片段过长/过短，调整静音间隔阈值
- 确保原始音频没有截断或静音

### Q: GPU 不可用？
- 检查是否安装 CUDA 驱动
- 确认 PyTorch CUDA 版本匹配
- 可切换到 CPU 模式继续使用

